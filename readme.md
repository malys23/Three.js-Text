# Three.js lil-gui Mesh and Texture Change Study <br>
Website where users can adjust text and experiment with different object shapes, density, and matcap colors and see the result. <br>

## Features
- Change the position, text, density of objects, matcap, object geometries <br>

## Tech Stack
- Three.js
- Node.js

Live Link: (https://three-js-text-dusky.vercel.app/)
